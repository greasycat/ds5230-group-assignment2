# Before starting

This project is built on top of the [SmartTutor],
You need to place the SmartTutor folder in the same directory as this project like this:

```
SmartTutor/
    ...
    ...
    ...
this_project/
    ...
    ...
    ...
```

Or you can edit the embeddign and meta path in the notebook.

# Set up environment

```bash
uv sync
```
